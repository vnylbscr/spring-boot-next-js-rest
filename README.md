# spring-boot-next-js-rest

A full stack rest app, you able to take notes, todos some stuffs. Written in Spring Boot (Java) and front end Next.js (React)

## Technologies

1. Spring Boot (Java)
2. React (Next.js, TypeScript, Chakra UI)
3. MongoDB
4. Heroku (Deployment)
